AI Quiz and Exam Generator System

 Description
This project is a Python-based AI system that generates exams automatically from a given topic. It includes multiple question types, timer, auto grading, and PDF export.

 Features
- Login system
- Exam generator (MCQ, True/False, Short Answer)
- Timer-based exam
- Student answer input
- Auto grading system
- Question bank storage
- Export exam to PDF

 How to Run
1. Install Python
2. Install library:
   pip install fpdf
3. Run:
   python ai_exam.py
